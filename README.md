# Coolculator

Coolculator is a custom calculator project built for [The Odin Project](https://www.theodinproject.com/).

The goal of the project was to build a working browser-based calculator using HTML, CSS, and JavaScript. The calculator supports basic arithmetic, chained operations, decimals, backspace, clear, keyboard input, and division-by-zero handling.

## Features

- Addition, subtraction, multiplication, and division
- Chained calculations
- Decimal input
- Backspace support
- Clear button
- Keyboard support
- Division-by-zero protection
- Rounded results to reduce long floating-point outputs
- Custom retro calculator design
- Hand-built layout using Flexbox

## Built With

- HTML
- CSS
- JavaScript

## What I Practiced

This project helped me practice:

- DOM selection and event listeners
- Separating calculator logic into functions
- Managing calculator state with variables
- Handling user input from both buttons and keyboard keys
- CSS layout using Flexbox
- CSS custom properties
- Styling a complete UI from scratch
- Debugging edge cases

## Keyboard Controls

| Key | Action |
|---|---|
| `0-9` | Enter numbers |
| `+` | Add |
| `-` | Subtract |
| `*` | Multiply |
| `/` | Divide |
| `Enter` or `=` | Calculate |
| `.` | Decimal |
| `Backspace` | Delete last digit |
| `Escape` or `Delete` | Clear |

## Notes

The extra function buttons such as `1/x`, `x²`, `²√`, and `±` are included visually as part of the calculator design, but they are not currently functional.
